# QA-testing
QA manual proyecto testing - plan de pruebas, casos y ejecución de pruebas y reporte de bugs.
Proyecto final hecho para el curso de Tester QA de ICARO. El mismo cuenta con plan de pruebas, casos y ejecución de pruebas, reporte de bugs y unos graficos de erorres. El mismo está hecho a raiz de un sitio web llamado Space & Beyond. El mismo se encarga de envios intergalacticos. Se probó principalmente modulos de Login/Logout, accesibilidad, buscador, filtrado, carrito y checkout. No se pudo probar el sistema de pagos ya que el sistema no cuenta con uno.
